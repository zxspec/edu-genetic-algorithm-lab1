# How to start

## Install dependencies
```sh
pip3 install -r requirements.txt
```

## Run to find solutions with a custom GA
```sh
python3 main.py
```

## Test GA with well know test functions
 ```sh
python3 test.py
```

## Visualization
 ```sh
python3 visualization/de_jong1.py
python3 visualization/rastrigin.py
```
